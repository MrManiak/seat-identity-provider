# SeAT Identity Provider

A SAML/OIDC identity provider plugin for SeAT (Simple EvE API Tool), a Laravel PHP application used for managing EvE Online corporations.

## Project Context

- **Framework**: Laravel (PHP)
- **Purpose**: Provides SAML authentication/identity provider functionality for SeAT
- **Parent Project**: SeAT - a web-based tool for managing EvE Online corporations and alliances
